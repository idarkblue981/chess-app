# Chess vs Stockfish AI

A modern desktop chess app, developed with Python and Pygame. It uses Stockfish chess engine.



## About
* **Cool interface:** Fullscreen window.
* **Lateral Panels:**
    * **Left:** Move history in algebraic chess notation.
    * **Right:** Panel for other settings.
* **Stockfish Engine:** ELO: 1400.
* **UI Design:** Move suggestions and highlite the last move.

## Used technologies
* **Python 3.x**
* **Pygame**
* **python-chess**
* **Stockfish Engine**
